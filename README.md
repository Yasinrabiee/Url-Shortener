## ✂️ Advanced Url Shortener

⚒️ Features:
- 📬 Sign-up
- 📭 Sign-in
- 🔐 Create secure link
- 🧊 Advanced AJAX list
- 📊 Status of the links
- 🎬 CURD operation on the links

⚡ Tech Stack:
- Styling: Fast Bootstrap (5.3)
- Language: PHP
- Database: Mysql (Maria DB)
- Template engine: eftec/bladeOne
- App router: bramus/router
- Architecture: MVC
